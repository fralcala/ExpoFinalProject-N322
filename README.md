# N322 Final

This app allows users to log in to create their own wishlist. Users can list an item they want then they can add a note or description. Only naming an item is necessary, users do note need to add a note if they don't want to. Users can edit their wishlist item name and the note/description. Users will need to be logged in to be able to add or edit their wishlist. Items can also be deleted if the user no longer want them on their list.

### Expo

`npx expo start`
or
`npx expo start --tunnel`
